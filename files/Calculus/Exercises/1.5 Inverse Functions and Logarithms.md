1) (10): **Determine if $f(x) = x^4 - 16$ is one-to-one:**

A function $f$ is one-to-one if and only if $f(x_1) \neq f(x_2)$ and $x_1 \neq x_2$. $f$ is not one-to-one because for example $f(4) = 240$ and $f(-4) = 240$. 

2) (25): **Find a formula for the inverse of $g(x) = 2 + \sqrt{x + 1}$**

$y = 2 + \sqrt{x + 1}$
$y - 2 = \sqrt{x + 1}$
$(y - 2)^2 =  x + 1$
$(y - 2)^2 - 1=  x$
$(x - 2)^2 - 1 =  y$

The inverse of $g(x)$ is $g^{-1}(x) = (x - 2)^2 - 1$
